# Data-Science-Internship-Task-2
# Titanic Survival Analysis Using Python

## Data Science Internship - Task 2

## 📌 Project Overview
This project analyzes the Titanic dataset to understand survival patterns based on passenger details like gender, class, and age.

## 🎯 Objectives
- Clean and preprocess Titanic data
- Handle missing values
- Perform Exploratory Data Analysis (EDA)
- Visualize survival patterns

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Analysis Performed
- Survival analysis by gender
- Survival rate by passenger class
- Age distribution analysis

## 📈 Key Insights
- Females had higher survival rates than males.
- First-class passengers had better survival chances.
- Age influenced survival probability.
